Miniprojet de Programmation

Description

Ce site a été réalisé par Daria Tupikina dans le cadre du cours Programmation et projet encadré, en préparation à un projet plus complexe portant sur l’étude de l’usage d’un mot sur Internet.

Période de réalisation du projet : 20/10/2025 – 17/11/2025

Étapes du projet

  1. écrire un programme Unix qui ouvre des liens à partir d’un fichier et affiche les informations nécessaires dans le Terminal ;
  2. modifier le programme pour que les informations soient enregistrées dans un tableau au format TSV ;
  3. modifier le programme pour que le tableau soit affiché sous forme de page HTML ;
  4. ajouter une page HTML contenant la description du mini-projet et un lien vers la page du tableau, modifier le style du site à l’aide de Bulma et publier le résultat sur GitHub Pages.
